# R&R — 역할과 책임 (Roles & Responsibilities)

> 기준 문서: [`실행계획.md`](./실행계획.md) (MVP, 4주). 스코프가 바뀌면 이 문서도 갱신.
> 한 줄: **개발 3인(승훈·종현·건·승연 중 승훈=PM)이 실행, 리나가 골든셋·발표 지원. 골든셋 정답은 PM이 최종 확정.**

---

## 1. 팀원별 책임

### 신승훈 — PM · 코어 · 통계
- **책임**: 일정·통합 관리 / 골든셋 정답 **최종 확정** / 채점기(scorer) / 통계(McNemar·부트스트랩) / 보고.
- **산출물**: `golden_master` 양식·라벨링 가이드, 채점기, 통계 결과, 보고서.
- **안 넘기는 선**: 골든셋 정답값은 PM이 확정(라벨러 의견 → 조정). 채점기는 **LLM 호출 없는 순수함수**.

### 노종현 — 온톨로지 · LLM 추출
- **책임**: 온톨로지(`trust_fund.ttl`·`shapes.ttl`) / 추출 프롬프트(tool-use) / JSON→ABox 변환 / 교차검증 로직 / 베이스라인(①) 모드.
- **산출물**: 온톨로지 2파일, 추출기, 교차검증, 그래프(W4).
- **안 넘기는 선**: 개념·속성 **이름의 단일 소스**. 변경 시 즉시 공지(가드·채점기가 깨짐).

### 조건 — 가드 · 실험
- **책임**: 가드 3종(형식·출처·제약) / 켜고끄기 스위치 / 변조 적용(코드) / 3조건 전수 실행 / 부트스트랩.
- **산출물**: 가드 모듈, 변조 적용 스크립트, 실험 결과 파일.
- **안 넘기는 선**: 가드는 **결정론 함수**(LLM 호출 금지). 실험은 동일 입력·고정 seed.

### 한승연 — 파이프라인 · CLI
- **책임**: 명령어 한 줄 파이프라인(PDF→추출→ABox→판정→채점→리포트) / 조건 전환 옵션 / 결과 표 집계 / 데모.
- **산출물**: CLI, 리포트 생성기, 실행법 README.
- **안 넘기는 선**: 모든 실행에 고유 `run_id`. 결과는 `reports/<run_id>/`에 저장.

### 김리나 — 골든셋 · 발표
- **책임**: 골든셋 라벨링(라벨러②) / 변조 케이스 **작성**(무엇을 바꿀지) / 발표 장표 · 문서.
- **산출물**: 라벨링 시트, 변조 케이스 표, 발표자료.
- **안 넘기는 선**: 코드 수정 금지. 추측 라벨 금지(모르면 `note` + PM 질문).

## 2. RACI 표

> **R**=실행 · **A**=최종책임(1명) · **C**=협의 · **I**=공유

| 산출물 / 활동 | 승훈 | 종현 | 건 | 승연 | 리나 |
|---------------|:---:|:---:|:---:|:---:|:---:|
| 온톨로지(ttl·shapes) | C | **A/R** | C | I | I |
| LLM 추출·tool-use | I | **A/R** | C | C | I |
| 교차검증 로직 | C | **A/R** | I | C | I |
| 가드 3종 | C | C | **A/R** | I | I |
| 파이프라인·CLI | I | C | C | **A/R** | I |
| 골든셋 라벨링 | **A** | I | C | I | **R** |
| 변조 케이스 작성 | A | I | C(적용) | I | **R** |
| 채점기(scorer) | **A/R** | C | I | C | I |
| 3조건 실험 실행 | A | C | **R** | C | I |
| 통계(McNemar·부트스트랩) | **A/R** | I | R | I | I |
| 발표 장표·보고서 | A(보고서) | C | C | C | **R**(장표) |
| 일정·통합 | **A/R** | I | I | I | I |

## 3. 협업 규칙 (최소)

- **(1) `main` 직접 push 금지** — 브랜치 → PR → 리뷰 1명 → 머지.
- **(2) 막히면 24시간 안에 PM 호출** — 혼자 끌어안지 않기.
- **인터페이스 먼저** — 추출 출력 형식 / 가드 결과 형식 / 골든셋 형식을 W1에 합의·고정한 뒤 병렬 작업.
- **보안** — `database/*.pdf`·`.env`는 커밋·외부 업로드 금지.
